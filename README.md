# DigitalDesign
Digital Design work 2025.2

Design a 3 bits to 3 bits circuit and PCB for add, subtract, multiply and XOR operation using AND, OR and NOT gate.

Notice that branch -main- contain clean schematic which doesn't optimized IC count for printing circuit, check branch -ic_optimized- for a pcb-ready schematic. It could be harder for routing but save space (printing PCB is expensive)
